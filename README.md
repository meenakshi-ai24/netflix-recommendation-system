# netflix-recommendation-system
This project focuses on analyzing Netflix movie data and building a basic
recommendation approach using Python and Jupyter Notebook.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Dataset
mymoviedb (public movie metadata used for educational purposes)

## Project Type
Data Analysis / Beginner Recommendation System

## Learning Source
This project was created for learning purposes during industrial training
and inspired by online tutorials.
